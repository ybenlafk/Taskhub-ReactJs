# TaskHub — React 19 + TypeScript + Vite

**TaskHub** is a task and project management app built with **React 19**, **TypeScript**, and **Vite**.

Create projects, manage tasks, and get a clear overview of your productivity with the built-in dashboard analytics.

### 🔗 Live Demo

👉 [View Demo](https://taskhub-pi.vercel.app/)

---

### ⚙️ Usage

You can use the provided Makefile to run and build the project:

```sh
make run     # Start the development server
make build   # Build the app for production
```

Or use the corresponding npm scripts:

```sh
npm run dev     # Start development server
npm run build   # Build for production
```
